Certifications & Skill Badges

Industry-recognized credentials to boost your resume and prove your skills, many are free or low-cost.

📊

Data & Analytics

Free Trial Available

DataCamp Certifications ↗

Python Data Associate, SQL Associate, Data Scientist, Data Engineer, and more. Timed assessments that result in shareable, verifiable credentials.

Paid

Power BI (PL-300) ↗

Microsoft's Power BI Data Analyst Associate certification, highly valued in business intelligence roles.

Paid

Tableau Certified Data Analyst ↗

Industry-standard Tableau certification that demonstrates visualization and analytics skills.

Free + Paid Tiers

Alteryx Fundamentals ↗

Credential in the popular data prep and analytics platform widely used in enterprise environments.

Free

KNIME Analytics ↗

Free courses and badges in the KNIME open-source data science platform. Good for no-code/low-code analytics.

☁️

Cloud & Infrastructure

Free Prep Available

AWS Cloud Practitioner ↗

The entry-level AWS cert, validates foundational cloud literacy. Excellent first cloud certification and very hireable.

Free Prep Available

Azure Fundamentals (AZ-900) ↗

Microsoft's foundational Azure cloud certification. Free practice exams are available via Microsoft Learn.

Free

Google Cloud Digital Leader ↗

Entry-level Google Cloud credential, great for demonstrating cloud fundamentals beyond AWS/Azure.

Free

Oracle Cloud Infrastructure Foundations ↗

Oracle's free cloud foundational cert. Less competitive to get, strong addition alongside AWS/Azure.

💻

Development & DevOps

Free

GitHub Foundations ↗

Official GitHub certification covering fundamentals of Git workflows, collaboration, and GitHub features.

Free

freeCodeCamp Certifications ↗

300-hour curriculum certifications in Responsive Web Design, JavaScript, Data Visualization, APIs, and more.

Free

Kaggle Micro-Certifications ↗

Quick, verifiable certificates in Python, Pandas, Machine Learning, Data Visualization, and SQL.

Free with financial aid

Google Data Analytics Certificate ↗

Industry-recognized Coursera cert from Google. Covers SQL, spreadsheets, R, and Tableau. Very hireable.

Free with financial aid

Meta Front-End Developer Certificate ↗

Build React projects and a portfolio. Issued by Meta, recognized by employers.

Free with financial aid

IBM Data Science Professional Certificate ↗

10-course series covering Python, SQL, ML, and data visualization. Good for breaking into data roles.

🛡️

Cybersecurity

Free Prep

CompTIA Security+ ↗

The most widely-recognized entry-level security cert. Required for many government and defense contractor roles.

Free

TryHackMe Learning Paths ↗

Completion badges and certificates for guided learning paths, great to show practical hands-on experience.

Paid

ISC² CC (Certified in Cybersecurity) ↗

Free to study, low exam cost, ISC² opened this entry-level cert to help diversify the cybersecurity workforce.

Which cert should I get first?

It depends on your goals. For data science/analytics, start with DataCamp or Kaggle. For cloud computing, try AWS Cloud Practitioner. For cybersecurity, look at CompTIA Security+ or ISC2 CC. For development, GitHub Foundations or freeCodeCamp are great starting points. When in doubt, ask in the Discord. Your GWC peers have been through this!
