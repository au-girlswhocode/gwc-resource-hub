Learn to Code

Beginner to advanced resources curated by GWC @ Augusta, web dev, data science, cybersecurity, tools, and more.

🌐

Web Development

HTML, CSS, JavaScript fundamentals, build your first websites and interactive apps.

🟢 Beginner

web.dev ↗

Modern tutorials on HTML, CSS, and performance best practices by Google.

🟢 Beginner → Advanced

freeCodeCamp ↗

Full learning paths for web development, data, and more. Completely free with certifications.

🟡 Intermediate

JavaScript30 ↗

30-day challenge to learn vanilla JavaScript through real projects. No frameworks needed.

🟢 Beginner

Codecademy ↗

Interactive tutorials for HTML, CSS, JS, and frameworks like React. Great for structured learners.

🟢 Beginner → Advanced

The Odin Project ↗

Full-stack web dev curriculum, completely free and project-based. One of the most complete self-taught paths available.

🟢 All Levels

MDN Web Docs ↗

Mozilla's official web documentation. The definitive reference for HTML, CSS, and JavaScript.

🟡 Intermediate

CSS-Tricks ↗

In-depth articles, guides, and tutorials on modern CSS and frontend techniques.

📊

Data Science & Analytics

Python, SQL, and BI tools for data-driven projects and analysis.

🟢 Beginner → Advanced

DataCamp ↗

Learn Python, SQL, Tableau, and Power BI with interactive notebooks. Also has certifications!

🟡 Intermediate

Kaggle Learn ↗

Free micro-courses on data visualization, machine learning, and AI tools from real practitioners.

🔵 Advanced

Pandas Docs ↗

Official Python library for data analysis and cleaning. The gold standard reference for data work.

🟢 Beginner

Google Data Analytics Certificate ↗

Google's popular Coursera certificate covering data cleaning, visualization, and SQL. Highly recognized.

🟢 All Levels

Towards Data Science ↗

Community publication with practical articles on data science, ML, and analytics.

🟢 Beginner

SQL Murder Mystery ↗

Learn SQL by solving an interactive murder mystery. Fun and actually effective.

🛡️

Cybersecurity & Networking

Ethical hacking, networking, and defensive security, check out the Cybersecurity page for more.

🟢 Beginner → Intermediate

TryHackMe ↗

Guided virtual labs for learning ethical hacking in a hands-on, gamified environment.

🟡 Intermediate → Advanced

Hack The Box ↗

Realistic penetration testing environments for sharpening real-world security skills.

🟢 Beginner

Coursera: Intro to Cybersecurity ↗

Learn core cybersecurity principles with practical labs through a structured, university-style path.

🧰

Tools & Version Control

Git, GitHub, and command-line tools used in real-world development.

🟢 Beginner

Learn Git Branching ↗

Interactive visual tutorials for mastering Git commands and branches. The most fun way to learn Git.

🟢 Beginner

Augusta Git Tutorial ↗

Student-made guide for version control basics, written specifically for AU students.

🟢 Beginner → Advanced

Exercism ↗

Over 60 languages with mentor feedback and test-driven challenges. Great for interview prep too.

🟢 Free

GitHub Student Developer Pack ↗

Free access to 100+ developer tools while you're a student. GitHub Copilot, domains, cloud credits, and more. Get this immediately.

🟢 All Levels

Roadmap.sh ↗

Community-built visual roadmaps for every major dev path — frontend, backend, DevOps, data science, etc.

🟢 Beginner

Replit ↗

Browser-based coding environment. No setup needed, great for quick projects and collaboration.

🤖

AI & Machine Learning

Explore the frontier of artificial intelligence and learn to build intelligent systems.

🟢 Beginner

fast.ai ↗

Free practical deep learning courses taught top-down, build real models before diving into theory.

🟡 Intermediate

Google ML Crash Course ↗

Google's free machine learning course with TensorFlow, structured, practical, and thorough.

🟢 Beginner

Hugging Face Learn ↗

Free NLP and AI courses, learn to use the latest transformer models and build AI applications.

🟡 Intermediate

Kaggle Competitions ↗

Real ML competitions with prizes and community notebooks. Great for portfolio building.

🟢 Beginner → Intermediate

DeepLearning.AI Short Courses ↗

Free short courses on LLMs, AI agents, and applied ML from Andrew Ng.

🔵 Advanced

Andrej Karpathy's YouTube ↗

Deep, hands-on explanations of how neural networks and LLMs actually work.

🧩

Problem Solving & Interview Prep

🟡 Intermediate

LeetCode ↗

Industry-standard platform for coding interview prep. Start with Easy problems and work up. Essential for tech internships.

🟢 Beginner → Intermediate

HackerRank ↗

Coding challenges with certifications. Many companies use it for hiring assessments.

🟡 Intermediate

NeetCode ↗

Curated LeetCode roadmap with video explanations. Much more approachable than grinding LeetCode blindly.

GitHub Repos Worth Bookmarking

free-programming-books ↗

The largest collection of freely available programming books and courses on the internet.

build-your-own-x ↗

Learn by rebuilding: databases, Git, Docker, HTTP server, and more from scratch.

project-based-learning ↗

Curated list of project-based tutorials across every major language and domain.

awesome-public-datasets ↗

Huge collection of free datasets for data science projects and competitions.

Developer Roadmap ↗

The roadmap.sh source repo. Thousands of stars, clear visual paths for every dev role.
